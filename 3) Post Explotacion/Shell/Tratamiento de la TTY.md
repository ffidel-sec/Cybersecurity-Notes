```bash
script /dev/null -c bash
```

2) Luego hacer ctrl + Z para poner en pausa la reverse shell

--------------------------------------

3) Desde TU terminal poner:

```bash
stty raw -echo;fg
```

----------------------------------

4) Luego es como que se poner en espera y poner:

```bash
reset xterm
```
