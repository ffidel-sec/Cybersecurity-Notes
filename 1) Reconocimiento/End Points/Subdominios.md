Enumerar un dominio con _SubFinder_:

```bash
subfinder -d dominio.com -o results.txt
```

Enumerar una lista de dominios:

```bash
subfinder -dL dominios.txt
```

Salida silenciosa:

```bash
subfinder -d dominio.com -silent
```
